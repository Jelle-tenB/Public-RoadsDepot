# RoadsDepot

## Description
I made an inventory app for Roads-Technology in Python using [Page 8.1](https://page.sourceforge.net/).
This program will help us track and update the inventory for [Roads Technology](https://roads-technology.nl/). It was made specifically for them and will most likely not be of any use to others, but I wanted to share it regardless.

The idea behind the app was to keep it as simple and intuitive as possible for easy use by any and all Roads Technology co-workers. That's why everything on the GUI is in Dutch.

There are a total of 5 accounts for logging purposes, which we later decieded to not use. 4 accounts have write permission and the other can only read.

## Public version
This version has been altered from the program in use, to prevent security breaches.

## Instructions
There is an .exe available for direct use but I will explain how to run the code aswell.

* Install the requirements (pip install -r requirements.txt).
* You can run either RoadsDepot2.py or RoadsDepot2_support.py.
* Use any of the following usernames:
    * "deelnemer1", "deelnemer2", "begeleider1" or "begeleider2" for full access.
    * "gebruiker" for a read-only account.
* Password is "Technology"
